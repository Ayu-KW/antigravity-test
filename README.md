# 寄付ランディングページ（外国人富裕層向け）

本プロジェクトは、日本の伝統文化や歴史的建造物の保護を目的とした、外国人富裕層向けの特別寄付プログラムを案内するランディングページです。

## ① 最初の命令文（要件）

> 寄付を募るLPを作成してほしい
> ターゲットは、外国人富裕層
> 旅行ついでに寄付をするような人たち
> 全体的な雰囲気は日本風な感じ

**【その他ユーザーからの指定条件】**
- 回答は常に日本語で行う
- HTMLを作成する際はセマンティックHTMLを基本とする
- CSSを作成する際は `destyle.css` をリセットCSSとして使用
- Webアクセシビリティに配慮した構造にする

## ③ プラン（実装方針）

外国人富裕層旅行者をターゲットとした、和風テイストのプレミアムな寄付募集ランディングページ（LP）を作成しました。「日本文化の保存・継承」などをテーマとし、直感的で心を動かす美しいデザインを目指しています。

### アプローチとデザイン方針
1. **デザイン美学**
   「漆（黒）」「金箔（ゴールド）」「真紅」などの伝統的な色彩をモダンにアレンジした、ダークモダンベースのエレガントなデザイン。
2. **Webアクセシビリティ**
   コントラスト比の確保、セマンティックなHTMLタグ（`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`）の使用、スクリーンリーダー対応（`aria-label`等）を徹底。
3. **寄付プランの設計**
   「松（MATSU）」「竹（TAKE）」「梅（UME）」の3つのティアを用意し、特別感のある特典を提示。

### ファイル構成
* `index.html`: セマンティックHTMLとアクセシビリティに配慮したメインページ。
* `destyle.css`: リセットCSS。
* `style.css`: 美しいグラデーション、スクロール連動のアニメーション、レスポンシブ対応を含むカスタムスタイル。
